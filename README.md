# Automated Cardiovascular Disease Diagnosis (ACVD-HBOMDL)

## About
A system for automated CVD diagnosis using Honey Badger Optimization 
(HBO) with a Modified Deep Learning model (DLMNN) and Bayesian optimization.

## Files
- 📄 ACVD_Report.pdf — Full project report
- 📊 ACVD_Presentation.pptx — Project presentation

## Tech Stack
- Python, Flask, SQLite3
- TensorFlow, Keras, Scikit-learn
- Pandas, NumPy, Matplotlib

## Algorithms Used
- Honey Badger Optimization (HBO)
- Stacking Classifier (RF + ET + LightGBM)
- Deep Learning Modified Neural Network (DLMNN)
- Bayesian Optimization
